pyEDAA.OutputFilter.CLI

Tools to extract data from log files.

Usage

pyedaa-outputfilter vivado-synthesis --file=toplevel.vds      --info     --info-file=info.log      --warning  --warning-file=warning.log      --critical --critical-file=critical.log      --error    --error-file=error.log      --influxdb-file synthesis.line

pyEDAA.OutputFilter.CLI.main()[source]

Entrypoint to start program execution.

Return type:

NoReturn

This function should be called either from:

  • if __name__ == "__main__": or

  • console_scripts entry point configured via setuptools in setup.py.

This function creates an instance of Program in a try ... except environment. Any exception caught is formatted and printed before the program returns with a non-zero exit code.
